# tic-tac-toe
This is a simple Tic Tac Toe game implemented in JavaScript using a command-line interface. The game can be played between two human players. 
This is a simple Tic Tac Toe game built with HTML, CSS, and JavaScript. The game can be played in the browser by two human players, and it features a minimalist design and basic animations.
Features Two-player mode: play against a friend or family member Simple design: focus on the gameplay, not the visuals Basic animations: see the X's and O's appear and disappear Responsive layout: the game should work on any device with a modern browser
How to use To play the game, simply open the index.html file in your favorite browser. The game board will appear, and you can click on any empty cell to place your mark (X or O). The game will automatically switch between the players and check for a winner or a tie.
